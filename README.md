## About

I build AI products and workflow automation — turning LLM and agent tooling into practical, reliable software. My work spans retrieval and orchestration pipelines, service automation, and full-stack prototypes with FastAPI and React. I hold an MS in Information Systems from Stevens Institute of Technology, with a focus on applied AI, systems design, and information systems.

- **Full-stack engineering** — FastAPI backends, React / TypeScript frontends
- **Information systems** — data modeling, SQL, and systems design
